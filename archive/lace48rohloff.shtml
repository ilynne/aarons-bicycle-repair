How to lace a Rohloff hub drilled for a 48 Hole Rim

Reference Point: Insert a spoke on each side of this hole.

It is not necessary to lace the radial spokes as shown above. Instead simply run the out-bound (a.k.a. heads-in) radial spokes along the outside of the crossed spokes.

Spoke length is 225mm Radial (16 spokes); 245mm Crossed (32 spokes): for 26" (559mm) Sun Rhyno Lite rim.

Spoke length is 249mm Radial (16 spokes); 269mm Crossed (32 spokes): for 700C (622mm) Velocity Dyad rim.

If you are using Damon Rinard's Spoke Calc, then enter 3.25 for the cross number.
